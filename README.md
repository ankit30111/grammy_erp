# grammy_erp

Custom Frappe app for Grammy Electronics ERP.

- Serves the Grammy (Lovable) single-page front-end same-origin at `/grammy`
  (uses the ERPNext login session - no tokens, no CORS).
- Holds Grammy-specific customizations (custom fields, doctypes, automations)
  upgrade-safely; ERPNext core is never modified.

Owned by ankit30111. Deployed via Frappe Cloud bench "New bench".
